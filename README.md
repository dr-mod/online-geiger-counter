# Online Geiger Counter

A device that keeps track of ionizing radiation.

![](docs/device.jpg)
### [Video in action](https://twitter.com/dr2mod/status/1584209364604588032)

### Hardware
* Raspberry Pi Pico W
* Gravity: Ionizing Radiation Detector
* SSD1306 OLED
